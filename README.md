# Apuntes de IA y Datos

Apuntes organizados en **55 unidades didácticas** distribuidas en 8 módulos.

| Módulo | Título | Horas |
|--------|--------|-------|
| MP01 | Procesamiento de datos para IA | 200 h |
| MP02 | Entrenamiento de modelos de aprendizaje automático | 190 h |
| MP03 | Desarrollo de componentes para sistemas de ML | 200 h |
| MP04 | Soluciones basadas en modelos de lenguaje (LLMs) | 160 h |
| MP01 | Implementación de sistemas de IA | 190 h |
| MP02 | Despliegue de sistemas de IA | 200 h |
| MP03 | Explotación de servicios de datos y analítica | 180 h |
| MP04 | Infraestructura para la ejecución de LLMs | 160 h |

Cada unidad incluye **slides** (presentación interactiva en HTML) y **temario** (contenido completo en Markdown).
